# /gen-post — Генерація поста для соцмереж

## Використання

```
/gen-post [тема] [платформа]
```

## Приклади

```
/gen-post "Урок кольорів" Instagram
/gen-post "Чому діти забувають слова" Facebook
/gen-post "Спарк показує алфавіт" TikTok
/gen-post "5 причин вчити англійську з 6 років" Facebook
```

## Правила для кожної платформи

### 📸 Instagram

- Довжина: 150–220 слів
- Починай з гачка — перше речення має зупинити скролінг
- Використовуй абзаци по 1–2 рядки (легко читати на телефоні)
- Емодзі: 6–10 штук, органічно вплетені в текст
- Заклик до дії в кінці: посилання в bio або “збережи цей пост”
- Хештеги: 15–20 штук окремим блоком після тексту
- Мова: українська + кілька англійських фраз природньо
- Тон: тепло, по-батьківськи, трохи захоплено

**Хештеги для English Quest (використовуй релевантні):**
`#англійськадлядітей #englishforkids #дитячаосвіта #вчимоанглійську #englishquest #діти69років #онлайннавчання #мамаукраїна #батькиукраїна #дошкільнята #початковашкола #englishlearning #kidsenglish #вчимосяграючи #розвитокдитини #освітаукраїна #мовнийрозвиток #спарк #дракончикспарк`

-----

### 👥 Facebook

- Довжина: 200–350 слів
- Починай з питання або провокаційного твердження
- Більш розгорнуто ніж Instagram — батьки на Facebook читають довше
- Можна додати міні-історію або кейс (“одна мама написала нам…”)
- Емодзі: 4–6 штук, стримано
- Заклик до дії: коментар, поширення або перехід на сайт
- Хештеги: 3–5 штук, тільки найважливіші
- Тон: авторитетний але дружній, як порада від досвідченого батька

-----

### 🎵 TikTok

- Формат: **сценарій відео** (не просто текст)
- Структура:
  - 0–3 сек: Hook — що зупинить скролінг (питання або шок-факт)
  - 3–15 сек: Розкриття теми (3 короткі пункти)
  - 15–25 сек: Демонстрація / Спарк в дії
  - 25–30 сек: Заклик до дії
- Підписи для відео (overlay text): виділи окремо
- Caption під відео: 80–120 символів + 5–8 хештегів
- Тон: енергійний, швидкий, захоплюючий

-----

### 📱 Загальні правила (всі платформи)

1. **Ніколи** не звучати як реклама — тільки цінність і користь
1. Спарк — живий персонаж, говори про нього як про реальну істоту
1. Конкретні цифри краще за абстракції (“24 уроки” а не “багато уроків”)
1. Якщо тема пов’язана з уроком — згадай конкретні слова або фрази з нього
1. Фінальний заклик до дії завжди веде на: https://english-quest-kids.netlify.app

Спочатку виведи:

```
📱 Платформа: [назва]
📌 Тема: [тема]
🎯 Мета: [охоплення / конверсія / залучення]
```

Якщо платформа — TikTok, додай окремо:

- Сценарій відео
- Overlay підписи
- Caption + хештеги

### Правила для промптів

**Стиль Спарка (використовуй коли він у сцені):**

```
cute friendly blue baby dragon named Spark,
big expressive round eyes, small book-shaped wings,
breathing colorful ABC letters, soft rounded body,
pink rosy cheeks, cheerful smile
```

**Загальний стиль курсу:**

```
children's book illustration style, soft pastel colors,
warm cozy atmosphere, 2D cartoon flat art,
bright and cheerful, educational theme,
clean white or light blue background
```

**Адаптуй промпт до теми посту:**

- Тема “кольори” → Спарк оточений кольоровими кулями або веселкою
- Тема “алфавіт” → Спарк видихає літери ABC у повітря
- Тема “числа” → Спарк тримає цифри як кульки
- Тема “тварини” → Спарк з маленькими тваринками навколо
- Тема “сім’я / батьки” → Спарк з дитиною і батьками
- Загальна тема → Спарк з книгою або в класі

**Тон зображення відповідає платформі:**

- Instagram → яскраве, eye-catching, з емоцією
- Facebook → тепле, сімейне, довіра
- TikTok → динамічне, контрастне, з рухом у сцені
